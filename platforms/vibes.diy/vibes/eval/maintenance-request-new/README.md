# eval-maintenance-request-new

> Maintenance request form for our apartment building. Tenants describe the problem and which unit they're in. The building manager sees all the requests and marks them in progress or done. Tenants only see their own.

Live at [https://vibes.diy/vibe/jchris/eval-maintenance-request-new](https://vibes.diy/vibe/jchris/eval-maintenance-request-new)

Single-file React app built with [vibes.diy](https://vibes.diy). Visit the live url to manage access.

## Run it

```sh
npx vibes-diy push     # uploads App.jsx, prints a live HTTPS URL
```

Edit [App.jsx](App.jsx) and push again to iterate.

## Commands

- `npx vibes-diy push` — deploy the current directory
- `npx vibes-diy push --instant-join` — deploy with auto-accept sharing
- `npx vibes-diy generate "prompt"` — generate a new app from a prompt
- `npx vibes-diy help` — full command list
